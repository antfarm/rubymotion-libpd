rubymotion-libpd
================

A skeleton RubyMotion [1] app that interacts with a Pure Data [2] patch [3] using pd-for-ios [4] and libpd [5].

To install libpd for iOS, execute ./bin/install_libpd.sh.
Then run rake to build the app.
